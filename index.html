User Information

Applying Operation

Method Explain

Converts text to all uppercase letters.
Example: "hello".toUpperCase() → "HELLO"

Converts text to all lowercase letters.
Example: "HELLO".toLowerCase() → "hello"

Splits string into an array by a separator.
Example: "Hello World".split(" ") → ["Hello", "World"]

Extracts part of a string.
Example: "Hello".slice(1,4) → "ell"

Reverses all letters of the string.
Example: "Hello".split("").reverse().join("") → "olleH"

Checks if string contains a substring.
Example: "Hello".includes("H") → true

Shows the number of characters in the string.
Example: "Hello".length → 5

Embed variables inside a string easily.
Example: `Hello ${name}` → "Hello Khadija"
